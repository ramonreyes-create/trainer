INSTRUCCIONES RÁPIDAS – DeutschQuest 7.0

QUÉ SUBIR A GITHUB:
Sube toda la carpeta DeutschQuest_7_0_Funcional completa.
No subas solo index.html.

CARPETAS NECESARIAS:
css
js
data
apps-script
docs

CONFIGURACIÓN GOOGLE SHEETS:
1. Crea una hoja nueva.
2. Extensiones > Apps Script.
3. Copia el archivo apps-script/Code.gs.
4. Implementar > Nueva implementación > Web app.
5. Ejecutar como: tú.
6. Acceso: cualquiera.
7. Copia la URL.
8. Pégala en js/config.js.

CURSO ABIERTO:
La alumna puede escribir cualquier curso o grupo.
Ejemplos: 7A, 8B, DSD II, Rodrigo, Intensivo B1.

ENVÍO:
Por defecto se envía todo al final.
Si quieres respaldo después de cada actividad, cambia en js/config.js:
SEND_MODE: "checkpoint"
