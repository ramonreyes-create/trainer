Lehrer-Modus · Google-Sheets-Setup für vollständige Berichte

Setup

1. Google-Sheets-URL eintragen

Trage hier die Web-App-URL ein, die mit /exec endet. Danach kannst du den fertigen Code in js/config.js kopieren.

Kurs ist in Version 7.0 kein fixes Menü mehr. Die Schülerin kann den Kurs frei schreiben.

2. Diesen Code in js/config.js einfügen

Wichtig: Speichern im Lehrer-Modus ändert GitHub nicht automatisch. Für GitHub Pages muss die Datei js/config.js wirklich geändert und hochgeladen werden.

3. Google Sheet

Das neue Apps Script erstellt automatisch zwei Blätter:

  • Resumen: eine Übersicht pro Sendung.
  • Detalle: jede einzelne Antwort, jeder Memory-Zug und jeder freie Text.

4. Das kommt auf GitHub

Lade den gesamten Inhalt dieser carpeta hoch, nicht den äußeren ZIP-Ordner.
